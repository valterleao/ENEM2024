DASHBOARD ENEM

Contexto:

O Exame Nacional do Ensino Médio (ENEM) é uma avaliação educacional de
ampla abrangência no Brasil, utilizada principalmente como critério de
acesso ao ensino superior. Os microdados do ENEM fornecem informações
detalhadas sobre os participantes, incluindo notas nas diversas áreas do
conhecimento, características socioeconômicas e dados demográficos.

Situação Problema:

Você é integrante do Ministério da Educação e precisa analisar os dados
do ENEM 2024 para tomar decisões acerca do futuro processo de aplicação
da prova, assim como para definir sobre outros programas educacionais
correlatos.

Objetivo da Tarefa:

Neste trabalho, você deverá construir um modelo OLAP para análise de
dados do ENEM 2024. A partir dos dados fornecidos no formato OLTP,
disponíveis no site do INEP, você será responsável por transformar esses
dados em um modelo multidimensional para posterior implementação no
Microsoft Power BI ou Tableau.

Base de Dados Pública:

Você usará os Microdados do ENEM, que estão disponíveis para download e
incluem uma vasta gama de informações sobre os participantes do exame. 
Base de Dados: Microdados do ENEM  Local de Acesso:
https://www.gov.br/inep/pt-br/acesso-a-informacao/dados
abertos/microdados/enem

**Modelo OLAP: **

Estruture um modelo multidimensional, definindo tabelas de fato e
dimensões. Identifique as principais medidas e dimensões relevantes para
a análise. 

O modelo deve permitir a análise de desempenho dos alunos com base em
aspectos como notas, características socioeconômicas, e localização
geográfica.

Dashboard em Microsoft Power BI ou Tableau: 

Crie um dashboard no Power BI ou Tableau utilizando o modelo OLAP
desenvolvido. O dashboard deve ser capaz de responder a perguntas
relacionadas ao desempenho dos alunos e outras questões relevantes. 

Questões a serem respondidas pelo Dashboard:

1\. Desempenho médio dos alunos por Estado e Região.

2\. Desempenho médio dos alunos por faixa socioeconômica.

3\. Distribuição das maiores notas de redação por estado e município.

4\. Distribuição geográfica dos alunos com as maiores notas por área de
conhecimento.

5\. Distribuição das notas por gênero e situação socioeconômica.

6\. Distribuição das notas por faixa etária.

7\. Distribuição das notas por raça/cor.

8\. Diferenças nas médias de notas entre alunos de capitais e do
interior.

9\. Comparação das notas por gênero em cada área de conhecimento.

10\. Média de notas entre alunos de escolas públicas e privadas por área
de conhecimento.
